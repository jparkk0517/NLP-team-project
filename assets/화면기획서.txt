[면접관 모드 화면기획서]

■ 화면명  
InterviewAgentChatPage  
(면접관 모드 채팅형 인터뷰 시뮬레이션 화면)

■ 목적  
입력된 자소서/JD 기반으로 생성된 면접 질문에 사용자가 답변하고,  
AI 면접관이 평가 및 후속 인터뷰 흐름을 유도하는 채팅 UI 기반 인터뷰 진행 화면

■ 주요 특징  
- 사용자와 면접관 간 실시간 대화형 채팅 형식
- 사용자는 답변을 텍스트 입력 가능
- 에이전트 응답 이후, 다음 인터뷰 흐름은 버튼 클릭으로 제어
- 흐름: [면접관 질문] → [사용자 답변] → [AI 평가] → [후속 버튼 선택]

━━━━━━━━━━━━━━━━━━━

■ 화면 구성 요소 (상단 → 하단 순)

0. [컨텍스트 업로드 영역]
───────────────────────
- 서버에 JD와 자소서가 업로드되어 있지 않을 경우 표시
- 구성 요소:
  ▸ 텍스트 입력 또는 파일 업로드 버튼: [자소서 업로드], [JD 업로드]
- 동작:
  ▸ 업로드 완료 시 "컨텍스트 저장 성공" 알림 후 채팅 UI 활성화
  ▸ 미업로드 시 채팅 영역 및 사용자 입력창 비활성화 또는 숨김
  ▸ 업로드 버튼을 통해 다시 컨텍스트 수정 가능

1. [헤더 영역]
───────────────────────
- 제목: "면접관 모드" (굵은 글씨)
- 설명: "입력한 자소서와 JD 기반으로 AI 면접을 시작합니다."
- 위치: 화면 최상단 중앙 정렬

2. [채팅 영역]
───────────────────────
- 구성: 좌우 대화형 채팅 UI
- 스크롤 가능 (상대방 메시지 순차 출력)
- 요소:
  └ 면접관 말풍선: 좌측 정렬, 말풍선 내 질문 텍스트  
  └ 사용자 말풍선: 우측 정렬, 입력된 답변 텍스트  
  └ 평가 메시지: 중앙 or 별도 회색 박스 형태로 출력

- 예시 대화 흐름:

  면접관:  
  👨‍💼 "자기소개서에서 협업 경험을 언급하셨는데, 구체적으로 어떤 역할을 하셨나요?"

  사용자:  
  🙋 "저는 팀에서 PM 역할을 맡아 일정 관리와 조율을 담당했습니다."

  AI 평가 (회색 박스):  
  �� 논리성: 보통  
  📊 구체성: 낮음  
  💡 개선 제안: 구체적인 수치나 사례 추가

3. [면접관 후속 선택 버튼] ← 중요
───────────────────────
- 출력 위치: AI 평가 메시지 **하단에 자동 출력됨** (입력창 아래 아님!!)
- 버튼 3개 나란히 정렬:

  [👉 꼬리질문 듣기]  
  [🌟 모범답변 보기]  
  [➡️ 다음 질문 받기]

- 버튼 기능:
  ▸ 꼬리질문 듣기 → 기존 질문 기반 후속 질문 채팅에 출력  
  ▸ 모범답변 보기 → AI 생성 예시 답변 채팅에 출력  
  ▸ 다음 질문 받기 → 다음 RAG 기반 질문 채팅에 출력

- 버튼 클릭 시:  
  ▸ 입력창은 자동으로 다시 활성화됨  
  ▸ 클릭된 버튼은 채팅창에 표시되지 않음 (면접관의 응답만 출력됨)

4. [사용자 입력 영역]
───────────────────────
- 구성 요소:
  ▸ 텍스트 입력창: "메시지 입력" (placeholder)  
  ▸ 버튼: [보내기]

- 동작 조건:
  ▸ 면접관 질문이 출력된 이후 활성화됨
  ▸ 사용자가 답변을 입력하고 보내기 클릭 시, 사용자 말풍선 출력
  ▸ 이후 텍스트 입력창 비활성화 → AI 평가 출력까지 잠시 대기

━━━━━━━━━━━━━━━━━━━

■ 인터랙션 흐름 요약 (1턴 예시)

① 면접관 질문 출력  
② 사용자 텍스트 입력 및 제출  
③ 사용자 답변 말풍선 출력  
④ AI 평가 결과 출력 (텍스트 or 박스)  
⑤ 3가지 후속 버튼 출력  
⑥ 버튼 클릭 → 새로운 면접관 메시지 출력  
⑦ 입력창 다시 활성화 → 다음 답변 대기

━━━━━━━━━━━━━━━━━━━

■ 예외 처리
───────────────────────
- 사용자가 아무것도 입력하지 않고 보내기 클릭 시 → "답변을 입력해 주세요" 알림
- 평가 출력 전에는 버튼이 출력되지 않음
- 꼬리질문, 모범답변, 다음질문은 한 턴에 하나만 선택 가능

━━━━━━━━━━━━━━━━━━━

■ 향후 확장 가능 항목
───────────────────────
- 답변 시간 측정 (타이머 표시)
- 음성 답변 입력(STT 연동)
- 답변 저장 기능 (리뷰 모드용)
- 면접 종료 후 전체 리포트(PDF 다운로드)
